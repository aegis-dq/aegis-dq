Data quality that tells you why

Aegis DQ is an open-source agentic framework that diagnoses pipeline failures, traces root causes, and proposes verified SQL fixes — automatically.

Get started → GitHub
pip install aegis-dq

How it works

1
Define rules in YAML 31 rule types — or run aegis generate to write them from your schema automatically.
2
Run the pipeline aegis run rules.yaml --db warehouse.duckdb — tables validate in parallel.
3
Get LLM diagnosis on every failure Plain-English explanation, root cause trace, and a verified SQL fix — for each failed rule.
4
Every decision is logged Full audit trail in SQLite — prompt, response, cost, latency. Searchable. Exportable.
